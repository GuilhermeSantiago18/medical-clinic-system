# medical-clinic-system  em CONSTRUÇÃO


Este é um projeto desenvolvido para conectar os médicos e profissionais da área aos seus pacientes, com a idéia principal de trazer facilidade para ambos.

## Funcionalidades
  * Profissional da saúde pode verificar sua  agenda diária/mensal.
  * Realizar agendamentos de novos pacientes.
  * Adicionar observações sobre motivo da consulta
  * Editar ou alterar algum agendamento ja feito.

# Orientações

<details>
  <summary><strong>Pré-requisitos</strong></summary>

  * NodeJS e NPM instalados
MongoDB instalado e rodando localmente ou em um servidor remoto



<details>
  <summary><strong>Como usar</strong></summary>

1. Clone o repositório em sua máquina
2. Abra o terminal e acesse a pasta do projeto
3. Acesse a pasta app/ e instale as dependências do frontend com o comando npm install
4. Ainda na pasta app/ execute o comando npm run dev 
5. Abra o seu navegador da web e vá para http://localhost:3000.

<details>
<summary><strong>Utilizando</strong></summary>
Ao acessar o frontend, o usuário irá se deparar com uma página de login, onde poderá digitar email e senha e acessar. Se o usuário for um médico ou profissional da área ele irá conseguir verificar toda sua agenda do dia, fazer alguma alteração caso deseje e verificar agenda mensal.
Caso o usuário seja o administrador é possível cadastrar novos pacientes, inserir novos horários e adicionar observações que devem ser vistas na tela do médico após a atualização.
